Team Features

Doc Pinning

Lock specific doc versions for the whole team. Every agent uses the same reference material.

chub pin openai/chat --lang python --version 4.0

Context Profiles

Role-scoped context with inheritance. Backend devs get API docs, frontend gets UI docs.

chub profile use backend

Agent Config Sync

Generate CLAUDE.md, .cursorrules, AGENTS.md from a single source of truth.

chub agent-config generate

Project Context

Custom markdown docs in .chub/context/ — architecture, conventions, runbooks — served alongside public docs.

chub get project/architecture

Dep Auto-Detection

Scan package.json, Cargo.toml, requirements.txt and auto-pin matching docs.

chub detect --pin

MCP Server

Stdio MCP server with team-aware tools. Agents get pinned versions, annotations, and project context.

chub mcp
